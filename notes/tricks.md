# 코딩 노트

## 임시변수 없이 변수 swap

```py
# 덧셈, 뺄샘을 통해 임시 변수 없이 swap
x = x + y
y = x - y
x = x - y

x, y = 9, 4
x = x + y # 13
y = x - y # 9
x = x - y # 4
```

```py
# XOR을 이용한 변수 swap
x = x ^ y
y = x ^ y
x = x ^ y

x, y = 9, 4 # 1001,0100
x = x ^ y   # 1001^0100 = 1101(13)
y = x ^ y   # 1101^0100 = 1001(9)
x = x ^ y   # 1101^1001 = 0100(4)
```
